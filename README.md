# 🏋️ Gym Management System

A web-based Gym Management System developed using Django to streamline gym operations and improve member management. The application provides a centralized platform for managing memberships, attendance, workout plans, trainers, and payments efficiently.

## 🚀 Features

* Member registration and profile management
* Membership plan creation and tracking
* Attendance monitoring system
* Trainer management
* Personalized workout plans
* Payment and billing management
* Admin dashboard for gym operations
* Secure user authentication and authorization

## 🛠️ Technologies Used

* Python
* Django
* HTML5
* CSS3
* JavaScript
* SQLite / PostgreSQL
* Django ORM

## 📊 Key Functionalities

### Member Management

* Add, update, and remove members
* Maintain member profiles and membership details

### Attendance Tracking

* Record daily attendance
* Monitor member activity and engagement

### Workout Management

* Create and assign workout plans
* Track fitness progress

### Payment Management

* Manage membership fees
* Track payment history and renewals

### Admin Dashboard

* View gym statistics and reports
* Manage members, trainers, and memberships from a single interface
